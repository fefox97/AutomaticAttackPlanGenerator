MACM Editor Information

The MACM Editor is a powerful tool that allows you to create and modify your MACM graphs directly within the application. With this editor, you can add nodes, define relationships, and customize properties to accurately represent your system architecture.

Key features of the MACM Editor include:

  • Node Management: Easily add, edit, and delete nodes representing various components of your system, by dragging elements from the palette onto the canvas.
  • Relationship Definition: Define connections between nodes to illustrate interactions and dependencies by clicking and dragging from one node to another.
  • Property Customization: Modify node and relationship properties to capture essential details.
  • Visual Representation: Utilize a user-friendly interface to visualize your MACM graph effectively.

Once your model is complete, simply click on the "Get Cypher" button to generate the Cypher query for your current MACM graph. This query can be used for further analysis or integration with other tools. In addition, you can upload the generated Cypher query directly into PenNet.

Keyboard shortcuts

⌘ + A: Select all nodes and relationships in the graph.
⌘ + Z: Undo the last action.
⌘ + Y: Redo the last undone action.
⌘ + C: Copy the selected node(s) or relationship(s).
⌘ + V: Paste the copied node(s) or relationship(s) at the current cursor position.
⌘ + X: Cut the selected node(s) or relationship(s).
↑ ← → ↓: Move the canvas view in the direction of the arrow key pressed.
Spacebar: Center the view on the selected node(s), if selected.
⌫ Remove the selected node or relationship from the graph.
